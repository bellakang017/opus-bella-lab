session protocol

The 4-Phase Session

Every collaboration session follows this rhythm. It keeps us fast and focused.

00:00–00:30

Phase 1 — Load Context

Paste the handoff file. I read it. We're synchronized in 30 seconds. No re-explanation needed.

00:30–02:00

Phase 2 — Define the Deliverable

State today's goal in one sentence. I'll ask 1–2 clarifying questions max, then propose a plan.

02:00–45:00

Phase 3 — Build

Deep work. I produce artifacts, you redirect as needed. No wasted loops. Todo list tracks progress.

45:00–50:00

Phase 4 — Close + Handoff

I generate an updated handoff file capturing everything we did. You save it for next session.

operating principles

Rules of Engagement

Bella leads direction. Opus leads structure.

You decide what matters and where we're going. I handle scaffolding, sequencing, quality control, and execution. Neither overrides the other.

One deliverable per session.

Focus creates quality. If a session tries to do too much, we split it. Better to ship one excellent artifact than three mediocre ones.

Everything gets versioned.

Files include dates and version numbers. Nothing is "final" — it's v1.0, v1.1, v2.0. This matches our growth mindset and keeps iteration fearless.

Close the loop before closing the tab.

Never end a session without saving the handoff. The 2 minutes you spend now saves 15 minutes next time. Compound efficiency.

upgrade path

Claude.ai Projects — Zero-Paste Mode

This is the most powerful upgrade for our workflow. Here's exactly how to set it up.

Step 1

Create a Project on claude.ai

Go to claude.ai → Projects → New Project → Name it "Opus × Bella"

Step 2

Add Project Knowledge

Upload the style guide HTML and your filled-in handoff template as "Project Knowledge" files. These persist across every conversation inside the project.

Step 3

Set Custom Instructions

In Project Settings → Custom Instructions, paste your user preferences (the P.A.G.E. protocol, research mentor role, etc.). This loads automatically in every chat.

Result

Every Conversation Starts Pre-Loaded

Open the project, type your goal, and I already know who we are, what we've built, and how to proceed. Zero paste. Maximum velocity.
